## 计算机组成原理A and 计算机组成原理与嵌入式系统 

计组A是未来技术拔尖班专业选修课，数学类（+计算机科学与技术双学士学位）必修考试课以及计算机与电子通信大类考试课。

**学分3.0 期末分数构成 30平时（包括实验+大报告+作业）+70考试（由100分期末试题折合而来）**

考试题目对于考察的同学来说，难度其实也相对较大，尤其在大二下这一课程扎堆的时期，复习时建议仔细的过一遍教材，然后仔细复习作业和老师大概率发的计组大总结。整体试卷方面全而细致，大题以作业原题，ppt原题和变式题目为主，认真听课并做了作业及格不难，考试课的同学建议在完整学习的基础上多多练习以冲击更高分，加油！
