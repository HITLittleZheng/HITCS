## 2.1 计算复杂性函数的阶

### 2.1.1 同阶函数集合 Θ（Theta）

**定义：** 设 $f(n)$ 和 $g(n)$ 是正值函数。若

$$\exists, c_1, c_2 > 0,; \exists, n_0,; \forall, n > n_0:\quad c_1 g(n) \le f(n) \le c_2 g(n)$$

则称 $f(n)$ 与 $g(n)$ **同阶**，记作 $f(n) = \Theta(g(n))$。

**集合表示：**

$$\Theta(g(n)) = {f(n) \mid \exists, c_1, c_2 > 0,; \exists, n_0,; \forall, n > n_0:; c_1 g(n) \le f(n) \le c_2 g(n)}$$

**核心含义：** $f(n)$ 在常数因子范围内与 $g(n)$ 增长速度相同（**紧界**）。

**经典例题：** 证明 $\frac{1}{3}n^2 - 3n = \Theta(n^2)$

> 需找 $c_1, c_2, n_0$ 使得 $c_1 n^2 \le \frac{1}{3}n^2 - 3n \le c_2 n^2$
>
> - **左侧：** 除以 $n^2$ 得 $c_1 \le \frac{1}{3} - \frac{3}{n}$，取 $c_1 = \frac{1}{6}$，当 $n > 18$ 时成立
> - **右侧：** 除以 $n^2$ 得 $\frac{1}{3} - \frac{3}{n} \le c_2$，取 $c_2 = \frac{1}{3}$，当 $n > 18$ 时成立
>
> 故取 $c_1 = 1/6,; c_2 = 1/3,; n_0 = 18$。证毕。

------

### 2.1.2 低阶函数集合 O（大O，上界）

**定义：** 设 $f(n)$ 和 $g(n)$ 是正值函数。若

$$\exists, c > 0,; \exists, n_0,; \forall, n > n_0:\quad f(n) \le c, g(n)$$

则称 $f(n)$ 比 $g(n)$ **低阶**（或 $g(n)$ 是 $f(n)$ 的**上界**），记作 $f(n) = O(g(n))$。

**集合表示：**

$$O(g(n)) = {f(n) \mid \exists, c > 0,; \exists, n_0,; \forall, n > n_0:; f(n) \le c, g(n)}$$

**核心含义：** $f(n)$ 的增长速度**不超过** $g(n)$（**渐近上界**）。

**例题：** $n = O(n^2)$，取 $c = 1, n_0 = 1$ 即可。

**重要概念：** 若 $f(n) = O(n^k)$，则称 $f(n)$ 是**多项式界限的**。

**命题 2.1.1：** 对于任意正整数 $d$ 和任意常数 $a_d \ne 0$，$d$ 次多项式 $p(n) = \sum_{i=0}^{d} a_i n^i = \Theta(n^d)$。

------

### 2.1.3 高阶函数集合 Ω（大Omega，下界）

**定义：** 设 $f(n)$ 和 $g(n)$ 是正值函数。若

$$\exists, c > 0,; \exists, n_0,; \forall, n > n_0:\quad f(n) \ge c, g(n)$$

则称 $f(n)$ 比 $g(n)$ **高阶**（或 $g(n)$ 是 $f(n)$ 的**下界**），记作 $f(n) = \Omega(g(n))$。

**集合表示：**

$$\Omega(g(n)) = {f(n) \mid \exists, c > 0,; \exists, n_0,; \forall, n > n_0:; f(n) \ge c, g(n)}$$

**核心含义：** $f(n)$ 的增长速度**不低于** $g(n)$（**渐近下界**）。

------

### 2.1.4 严格低阶函数集合 o（小o，严格上界）

**定义：** 设 $f(n)$ 和 $g(n)$ 是正值函数。若

$$\forall, c > 0,; \exists, n_0,; \forall, n > n_0:\quad f(n) < c, g(n)$$

则记作 $f(n) = o(g(n))$。

**与大O的区别：**

|        | 大O              | 小o                  |
| ------ | ---------------- | -------------------- |
| 量词   | $\exists, c > 0$ | $\forall, c > 0$     |
| 不等号 | $\le$            | $<$                  |
| 含义   | 上界（可能紧）   | 严格上界（一定不紧） |

**等价条件：** $f(n) = o(g(n)) \iff \lim_{n\to\infty} \frac{f(n)}{g(n)} = 0$

------

### 2.1.5 严格高阶函数集合 ω（小omega，严格下界）

**定义：** 设 $f(n)$ 和 $g(n)$ 是正值函数。若

$$\forall, c > 0,; \exists, n_0,; \forall, n > n_0:\quad f(n) > c, g(n)$$

则记作 $f(n) = \omega(g(n))$。

**等价条件：** $f(n) = \omega(g(n)) \iff \lim_{n\to\infty} \frac{f(n)}{g(n)} = \infty$

------

### 2.1.6 五种记号总结对比

| 记号            | 含义         | 类比      | 量词要求            |
| --------------- | ------------ | --------- | ------------------- |
| $f = \Theta(g)$ | 同阶（紧界） | $f = g$   | $\exists, c_1, c_2$ |
| $f = O(g)$      | 上界         | $f \le g$ | $\exists, c$        |
| $f = \Omega(g)$ | 下界         | $f \ge g$ | $\exists, c$        |
| $f = o(g)$      | 严格上界     | $f < g$   | $\forall, c$        |
| $f = \omega(g)$ | 严格下界     | $f > g$   | $\forall, c$        |

### 2.1.7 函数阶的性质

1. **传递性：** 五种记号全部满足传递性
   - 若 $f = O(g)$ 且 $g = O(h)$，则 $f = O(h)$（Θ, Ω, o, ω 同理）
2. **自反性：** $\Theta$、$O$、$\Omega$ 满足自反性
   - $f = \Theta(f)$，$f = O(f)$，$f = \Omega(f)$
3. **对称性：** 只有 $\Theta$ 满足对称性
   - $f = \Theta(g) \iff g = \Theta(f)$
4. **转置对称性：**
   - $f = O(g) \iff g = \Omega(f)$
   - $f = o(g) \iff g = \omega(f)$
5. **$\Theta$ 与 $O$、$\Omega$ 的关系：**
   - $f = \Theta(g) \iff f = O(g)$ 且 $f = \Omega(g)$

**⚠️ 注意：不是所有函数都可比！**

> 例：$f(n) = n$ 与 $g(n) = n^{1+\sin n}$ 不可比，因为 $\sin n$ 在 $-1$ 和 $1$ 之间振荡。

## 2.2 标准符号和通用函数

### 2.2.1 Floor 和 Ceiling

**定义：**

- $\lfloor x \rfloor$（Floor）：小于或等于 $x$ 的**最大整数**
- $\lceil x \rceil$（Ceiling）：大于或等于 $x$ 的**最小整数**

**基本性质：**

- $x - 1 < \lfloor x \rfloor \le x \le \lceil x \rceil < x + 1$
- 对整数 $n$：$\lfloor n/2 \rfloor + \lceil n/2 \rceil = n$

## 2.3 和式的估计与界限

### 2.3.1 线性和

$$\sum_{k=1}^{n}(c \cdot a_k + b_k) = c\sum_{k=1}^{n}a_k + \sum_{k=1}^{n}b_k$$

### 2.3.2 常用级数公式

| 名称     | 公式                                                         |
| -------- | ------------------------------------------------------------ |
| 等差级数 | $\displaystyle\sum_{k=1}^{n} k = \frac{n(n+1)}{2} = \Theta(n^2)$ |
| 平方和   | $\displaystyle\sum_{k=1}^{n} k^2 = \frac{n(n+1)(2n+1)}{6}$   |
| 几何级数 | $\displaystyle\sum_{k=0}^{n} x^k = \frac{x^{n+1}-1}{x-1}$，当 $x \ne 1$ |
| 调和级数 | $\displaystyle\sum_{k=1}^{n} \frac{1}{k} = \ln n + O(1)$     |
| 对数和   | $\displaystyle\sum_{k=1}^{n} \lg k = \Theta(n \lg n)$        |

### 2.3.3 求和的界限估计方法

- **积分近似法：** 对单调递增函数 $f(k)$：
  $$\int_0^n f(x),dx \le \sum_{k=1}^{n} f(k) \le \int_1^{n+1} f(x),dx$$

## 2.4 递归方程

### 2.4.1 什么是递归方程

**定义：** 用具有较小输入值的同一方程来描述方程本身。

**经典例子 —— 归并排序：**

$$T(n) = \begin{cases} \Theta(1) & n = 1 \ 2T(n/2) + \Theta(n) & n > 1 \end{cases}$$

解为 $T(n) = \Theta(n\log n)$。

------

### 2.4.2 Substitution 方法（替换法/代入法）

**基本步骤：**

1. **猜测**解的形式
2. 用**数学归纳法**证明猜测正确

#### 技巧I：联想已知的 $T(n)$

**例：** 求解 $T(n) = 2T(\lfloor n/2 \rfloor + 17) + n$

> 联想到类似 $T(n) = 2T(n/2) + n$ 的解是 $\Theta(n\log n)$
> 猜测 $T(n) = O(n\log n)$，用归纳法证明

#### 技巧II：逐步缩小猜测范围

> - 先证下界：若已知 $T(n) = \Omega(n)$，上一阶尝试 $\Omega(n\log n)$
> - 再证上界：若已知 $T(n) = O(n^2)$，下一阶尝试 $O(n\log n)$
> - 两者吻合则 $T(n) = \Theta(n\log n)$

#### 技巧III：变量替换

将递归方程通过变量替换转化为熟悉的形式。

**例：** 求解 $T(n) = 2T(\sqrt{n}) + \lg n$

> 令 $n = 2^m$，则 $T(2^m) = 2T(2^{m/2}) + m$
> 设 $S(m) = T(2^m)$，得 $S(m) = 2S(m/2) + m$
> 解得 $S(m) = \Theta(m\log m)$
> 回代：$T(n) = \Theta(\lg n \cdot \lg\lg n)$

#### ⚠️ 常见陷阱

证明 $T(n) = O(n)$ 时，不能从 $T(n) \le cn + n$ 得到 $T(n) \le cn$。

**解决方法：** 从猜测中**减去一个低阶项**。例如猜 $T(n) \le cn - b$。

------

### 2.4.3 Iteration 方法（迭代展开法）

**基本步骤：**

1. 循环展开递归方程
2. 将递归方程转化为**和式**
3. 使用求和技术求解

**例：** $T(n) = 2T(n/2) + n$，$T(1) = 1$

$$T(n) = 2T(n/2) + n$$
$$= 2[2T(n/4) + n/2] + n = 4T(n/4) + 2n$$
$$= 4[2T(n/8) + n/4] + 2n = 8T(n/8) + 3n$$
$$= \cdots$$
$$= 2^k T(n/2^k) + kn$$

当 $n/2^k = 1$，即 $k = \log_2 n$ 时：

$$T(n) = nT(1) + n\log n = n + n\log n = \Theta(n\log n)$$

------

### 2.4.4 Master 方法（主定理）

#### 适用形式

$$T(n) = aT(n/b) + f(n)$$

其中 $a \ge 1$，$b > 1$ 为常数，$f(n)$ 为渐近正函数。

**直观理解：** 将大小为 $n$ 的问题分成 $a$ 个大小为 $n/b$ 的子问题，合并代价为 $f(n)$。

#### Master 定理（三种情况）

**比较 $f(n)$ 与 $n^{\log_b a}$ 的大小关系：**

| 情况       | 条件                                                         | 结论                                |
| ---------- | ------------------------------------------------------------ | ----------------------------------- |
| **Case 1** | $f(n) = O(n^{\log_b a - \varepsilon})$，某 $\varepsilon > 0$ | $T(n) = \Theta(n^{\log_b a})$       |
| **Case 2** | $f(n) = \Theta(n^{\log_b a})$                                | $T(n) = \Theta(n^{\log_b a} \lg n)$ |
| **Case 3** | $f(n) = \Omega(n^{\log_b a + \varepsilon})$，某 $\varepsilon > 0$，且 $af(n/b) \le cf(n)$，某 $c < 1$ | $T(n) = \Theta(f(n))$               |

**核心思想：** 比较**递归树叶子代价** $n^{\log_b a}$ 与**合并代价** $f(n)$ 的大小：

- Case 1：叶子代价占主导 → 答案由叶子决定
- Case 2：两者相当 → 乘以 $\lg n$
- Case 3：合并代价占主导 → 答案由 $f(n)$ 决定

#### ⚠️ 关键要求："多项式地大于/小于"

$f(n)$ 必须与 $n^{\log_b a}$ 相差一个**多项式因子** $n^\varepsilon$。若只差一个对数因子（如 $f(n) = n^{\log_b a} \lg n$，非 Case 2 的形式），则 Master 定理**不适用**。

#### 使用示例

**例1：** $T(n) = 9T(n/3) + n$

> $a=9, b=3, f(n)=n, n^{\log_b a} = n^{\log_3 9} = n^2$
> $f(n) = n = O(n^{2-\varepsilon})$，取 $\varepsilon = 1$ → **Case 1**
> $T(n) = \Theta(n^2)$

**例2：** $T(n) = T(2n/3) + 1$

> $a=1, b=3/2, f(n)=1, n^{\log_b a} = n^{\log_{3/2} 1} = n^0 = 1$
> $f(n) = \Theta(1) = \Theta(n^{\log_b a})$ → **Case 2**
> $T(n) = \Theta(\lg n)$

**例3：** $T(n) = 3T(n/4) + n\lg n$

> $a=3, b=4, n^{\log_b a} = n^{\log_4 3} \approx n^{0.793}$
> $f(n) = n\lg n = \Omega(n^{0.793+\varepsilon})$，取 $\varepsilon \approx 0.2$ → 检查正则条件 → **Case 3**
> $T(n) = \Theta(n\lg n)$

**反例（不适用）：** $T(n) = 2T(n/2) + n\lg n$

> $a=2, b=2, n^{\log_b a} = n$
> $f(n) = n\lg n$ 不是多项式地大于 $n$（只大了 $\lg n$ 因子）
> **Master 定理不适用！**

------

### 2.4.5 Master 定理证明思路（了解）

**证明框架（$n = b^k$ 情况）：**

**第一步**（展开递归树）：

$$T(n) = n^{\log_b a} \cdot T(1) + \sum_{j=0}^{\log_b n - 1} a^j f(n/b^j)$$

令 $g(n) = \sum_{j=0}^{\log_b n - 1} a^j f(n/b^j)$

**第二步**（分三种情况估计 $g(n)$）：

- Case 1: $g(n) = O(n^{\log_b a})$
- Case 2: $g(n) = \Theta(n^{\log_b a} \lg n)$
- Case 3: $g(n) = \Theta(f(n))$

**第三步**（合并得到 $T(n)$ 的渐近界）

对 $n \ne b^k$ 的一般情况，需处理 ceiling/floor，但最终结论不变。

### 2.4.6 生成函数方法（了解）

**适用场景：** 算法平均情形分析。

**基本步骤：**

1. 递归两边乘 $z^n$，对 $n$ 求和
2. 得到关于生成函数 $A(z)$ 的方程
3. 解方程得 $A(z)$ 的显式
4. 展开为幂级数，提取系数 $a_n$

**简单例子：** $T(n) = T(n-1) + 1$，$T(0) = 0$

> 得到 $A(z) = \frac{z}{(1-z)^2}$
> 展开得 $a_n = n$，即 $T(n) = n$
