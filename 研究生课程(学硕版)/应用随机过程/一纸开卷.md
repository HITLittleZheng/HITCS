(1) 证明 $F(x,y)$ 对每个变元非降．
证明：首先证明 $F(x,y)$ 对于 $x$ 非降．固定 $y=\hat{y}$．

1° 若 $x_1 + \hat{y} < x_2 + \hat{y} < 0$，则有 $F(x_1,\hat{y})=0 \leq F(x_2,\hat{y})=0$．
2° 若 $0 \leq x_1 + \hat{y} < x_2 + \hat{y}$，则有 $F(x_1,\hat{y})=1 \leq F(x_2,\hat{y})=1$．
3° 若 $x_1 + \hat{y} < 0 \leq x_2 + \hat{y}$，则有 $F(x_1,\hat{y})=0 \leq F(x_2,\hat{y})=1$．
综上三种情况，可以证明 $F(x,y)$ 对于 $x$ 非降．

综上两种情况, 可以证明 $F(x, y)$ 关于 $x$ 右连续．

**26.**分布函数再生性问题:
若 $ξ₁ \sim B(m, p), ξ₂ \sim B(n, p)$, 且 $ξ₁$ 与 $ξ₂$ 独立, 则 $η = ξ₁ + ξ₂ \sim B(m + n, p)$.

已知$\xi_1\sim B(m,p)$，其概率质量函数为$P(\xi_1 = k)=\binom{m}{k}p^{k}(1 - p)^{m - k}$，
$k = 0,1,\cdots,m$；
$\xi_2\sim B(n,p)$，其概率质量函数为$P(\xi_2 = j)=\binom{n}{j}p^{j}(1 - p)^{n - j}$，$j = 0,1,\cdots,n$ 。
因为$\eta=\xi_1+\xi_2$，对于离散型随机变量，$\eta$的概率质量函数$P(\eta = l)$可由卷积公式
$P(\eta = l)=\sum_{k = 0}^{l}P(\xi_1 = k)P(\xi_2 = l - k)$
（这里$l$的取值范围是$0\leq l\leq m + n$ ）计算。将$\xi_1$和$\xi_2$的概率质量函数代入卷积公式：
$P(\eta = l)=\sum_{k = 0}^{l}\binom{m}{k}p^{k}(1 - p)^{m - k}\cdot\binom{n}{l - k}p^{l - k}(1 - p)^{n-(l - k)}$

$=\sum_{k = 0}^{l}\binom{m}{k}\binom{n}{l - k}p^{k+(l - k)}(1 - p)^{m - k + n-(l - k)}$

$=p^{l}(1 - p)^{m + n - l}\sum_{k = 0}^{l}\binom{m}{k}\binom{n}{l - k}$

所以$P(\eta = l)=\binom{m + n}{l}p^{l}(1 - p)^{m + n - l}$，$l = 0,1,\cdots,m + n$ 。
这正是二项分布$B(m + n,p)$的概率质量函数形式。
