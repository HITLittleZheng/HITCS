实验（三）深度学习图像复原（DnCNN）、传统图像分割与医学图像分割（U-Net）
1. 实验名称
深度学习图像复原（DnCNN）、传统图像分割与医学图像分割（U-Net）

3. 实验目的
1)	理解深度卷积神经网络在低层视觉任务中的基本应用流程，掌握图像去噪任务中的退化建模、网络推理、结果评价与参数分析方法；
2)	理解 DnCNN 的残差学习思想，掌握其通过学习噪声残差实现图像复原的基本原理；
3)	理解传统图像分割算法的基本思想，掌握阈值分割、Otsu 自适应阈值、Canny 边缘检测、轮廓检测、K-means 聚类分割、区域生长、Watershed 与 Mean Shift 等方法的适用场景；
4)	掌握传统分割算法中阈值、邻域、种子点、聚类数、形态学操作和距离变换等关键参数对结果的影响；
5)	理解医学图像语义分割的基本任务形式，掌握 U-Net 编码器—解码器结构、跳跃连接和像素级预测的作用；
6)	掌握分割任务中 IoU、Accuracy 等评价指标的计算方法，能够结合预测图、标签图和定量指标分析模型效果；
7)	比较传统图像处理方法与深度学习方法在可解释性、参数依赖、数据依赖、泛化能力和失败案例上的差异。

5. 实验内容与进度安排
序号	实验任务	知识点
1	任务一：DnCNN 图像去噪环境配置与代码阅读。进入 KAIR/ 目录，查看 main_test_dncnn.py、models/network_dncnn.py、model_zoo/ 和 testsets/ 的作用，明确输入图像、预训练权重、输出结果和日志文件之间的关系。	图像复原、预训练模型、PyTorch 推理流程、模型参数加载
2	任务二：DnCNN 去噪实验运行与参数对比。使用 dncnn_15、dncnn_25、dncnn_50 等模型，在 set5、set12 或 bsd68 测试集上运行去噪程序，保存噪声图、复原图和原图，并记录 PSNR、SSIM。	高斯噪声、噪声强度 σ、PSNR、SSIM、残差学习
3	任务三：传统图像分割代码阅读与运行。进入 Classical-Image-Segmentation/ 目录，阅读 main.py 以及各分割算法脚本，理解输入图像、测试标签、结果保存目录和多算法可视化流程。	阈值分割、Otsu、Canny、轮廓检测、K-means、区域生长、Watershed、Mean Shift
4	任务四：传统分割结果比较与参数分析。运行 main.py，对 test_data/ 中的微生物图像进行分割，观察 segmentation_results/ 中不同算法的输出差异；修改至少两类算法的关键参数，分析参数变化对分割结果的影响。	二值化、边缘与区域、聚类分割、种子点选择、距离变换、过分割与欠分割
5	任务五：U-Net 医学图像分割代码阅读。进入 Unet/ 目录，查看 network.py、test_segmentation.py、checkpoint/UNET_model.pth 和 dataset/ 的结构，理解 X-ray 图像、mask 标签和模型输出之间的关系。	语义分割、U-Net、编码器—解码器、跳跃连接、二值 mask
6	任务六：U-Net 分割推理与指标计算。运行 test_segmentation.py，对测试集 X-ray 图像进行分割，保存 output/ 中的拼接结果图，并记录每张图像的 IoU 与 Accuracy。	DataLoader、Sigmoid 输出、阈值化、IoU、Accuracy
7	任务七：复原、传统分割与深度分割综合比较。比较 DnCNN 去噪、传统图像分割和 U-Net 医学分割的输入输出、模型假设、参数依赖、评价指标和典型失败案例，完成实验报告。	视觉任务建模、传统方法与深度学习方法比较、实验报告撰写
6. 实验要求

7. 环境配置与运行方式（参考课上视频附件，以下内容仅供参考）

DnCNN 图像去噪任务：
常用参数示例：
python main_test_dncnn.py --model_name dncnn_25 --testset_name set5 --noise_level_img 25 --device cpu
python main_test_dncnn.py --model_name dncnn_50 --testset_name set12 --noise_level_img 50 --device cpu
输出结果参考：
KAIR/results/<testset_name>_<model_name>/
其中通常包含：
*_L.*  噪声图像
*_E.*  DnCNN 复原图像
*_H.*  原始高质量图像
*.log  PSNR/SSIM 日志

传统图像分割任务：
cd /root/prj/exp-03/Classical-Image-Segmentation
pip install -r requirement.txt
python main.py
输出结果参考：
Classical-Image-Segmentation/segmentation_results/

U-Net 医学图像分割任务：
python test_segmentation.py
输出结果参考：
Unet/output/
每张输出图通常由三部分横向拼接组成：
X-ray 原图 | U-Net 预测 mask | 人工标注 mask

分析提示：
•	DnCNN 为什么采用残差学习而不是直接预测干净图像？
•	噪声强度 σ 与 PSNR/SSIM 之间通常有什么关系？
•	固定阈值分割和 Otsu 分割分别适合什么样的灰度分布？
•	Canny 边缘检测得到的是边界信息，为什么它不一定直接等价于完整目标区域？
•	K-means 聚类数 K 过大或过小会怎样影响分割结果？
•	区域生长算法为什么对种子点位置和相似性阈值敏感？
•	Watershed 为什么容易出现过分割？可以用哪些预处理或 marker 策略缓解？
•	U-Net 中跳跃连接为什么有助于恢复边界和细节？
•	IoU 与 Accuracy 的侧重点有什么不同？在前景区域很小的分割任务中，为什么只看 Accuracy 可能产生误判？
•	传统图像处理方法和深度学习方法在可解释性、数据依赖、可迁移性和部署成本方面各有什么优势与局限？
